## TockyClock

TockyClock is a clock app that helps you stay focused on a task. 

### But why?

The ticking noise helps me stay focused on a task. It's a constant reminder to not do other stuff when I'm supposed to be working. 

### Features

* ticking
* alarm
* responsive design

### Built using

* html
* css
* love
* sweat
* tears
* vanilla.js
